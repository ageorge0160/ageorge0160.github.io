This site is a way for me to keep a collection of team building activities. As a facilitator of team building and leadership initiatives, I have many activities, games and initiatives to keep track of. This site keeps games listed from all users and each user. Game views will show the full description of a single game.

Instructions

Fork and clone the repository. Run bundle install and rackup.

To use app, create a username and log in. Begin adding activities.
